# Contributors

Thank you to everyone who contributes to Lexi Translator!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Follow the coding guidelines in `.kiro/steering/coding-guidelines.md`
4. Ensure all files include the AGPL license header
5. Run `./gradlew ktlintCheck detekt` before committing
6. Submit a Pull Request with a clear description
